# Employee Attendance Management System

A cloud-based employee attendance management system built using Google Apps Script, HTML, CSS, and JavaScript.

## Features

* Employee registration and login
* Attendance recording with geofencing
* Password reset
* Session timeout
* User-friendly interface

## Requirements

* Google Apps Script
* Google Sheets
* HTML
* CSS
* JavaScript

## Installation

1. Create a new Google Apps Script project.
2. Create a new Google Sheets spreadsheet.
3. Copy the code from this repository into your Google Apps Script project.
4. Configure the script to use your Google Sheets spreadsheet.

## Usage

1. Employees can register and log in to the system.
2. Employees can record their attendance by clicking on the "Check-In" or "Check-Out" buttons.
3. The system checks for geofencing and ensures that the employee is within the designated area.
4. Attendance records are stored in the Google Sheets spreadsheet.
